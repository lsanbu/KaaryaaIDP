## 4️⃣ Consent Handling – How to Do It Safely (You’re Thinking Correctly)

Your instinct here is good, but let’s frame it correctly.

You should NOT:

* Manage user consent
* Store consent records as a system of record
* Replace CMPs

What you CAN do safely:

### Evidence-Aware Consent Capture

* Capture **declared consent date**
* Capture **consent source (application form / checkbox / declaration)**
* Capture **document-embedded consent text**
* Include as **metadata in output**

Example:

```json
"consent": {
  "declared": true,
  "source": "Physical application form",
  "captured_date": "2025-01-18",
  "evidence_type": "Document declaration",
  "confidence": 0.82
}
```

This is **not consent management**.
This is **consent evidence extraction**.

Very different — and very safe.
